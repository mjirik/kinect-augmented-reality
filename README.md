Spuštění aplikace 
========================

Kalibraci spustíme prostřednictvím souboru kalibrace2.py. Výstupem toho programu je kalibrační matice.

    python kalibrace2.py



Pro spuštění samotné aplikace slouží soubor sledovani.py s obrazovým výstupem.

    python sledovani.py
